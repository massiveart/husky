Provided Events

Direction Event Name Parameter Description
in husky.button.<<instancename>>.click --- button where clicked
out husky.button.<<instancename>>.state string|array ... typeName or type array change buttonType [icon, custom type]

Used Events

Event Name Description
No event used

Options

Option Name Type Default Description
buttontype string 'icon' type of button [icon, custom]
instancename string 'undefined' name of instance used in events
text string 'undefined' button text
icontype string 'caution' button icon
background string null background color for spinner

Types and States

Type State Description
icon Create a button with an icon and a text
standard Button is active
disable Button is disabled
loading Button is not visible, but a loading icon appears
