Provided Events

Direction Event Name Parameter Description
in husky.header.move-buttons int ... navigation width move middle part to match given navigation width
in husky.header.button-type string|array ... typeName or type array change buttonType [save, saveDelete, add, custom type]
in husky.header.button-state string ... name of state change state of buttons

Used Events

Event Name Description
navigation.item.content.show used to move buttons when the content is changing

Options

Option Name Type Default Description
marginmid int 45 add 45px to navWidth
marginright int 20 add 20px to margin mid part
buttontype string null type of button at start

Types and States

Type State Description
save Show a SaveButton in the left part
standard SaveButton is active
disable SaveButton is disabled
loading-save-button SaveButton is not visible, but a loading icon appears
saveDelete Show a SaveButton in the left part and a DeleteButton in the right part
standard SaveButton and DeleteButton is active
disable SaveButton and DeleteButton is disabled
loading-save-button SaveButton is not visible, but a loading icon appears and the DeleteButton is disabled
loading-delete-button DeleteButton is not visible, but a loading icon appears and the SaveButton is disabled
add Show a AddButton in the left part
standard AddButton is active
disable AddButton is disabled
loading-save-button AddButton is not visible, but a loading icon appears
