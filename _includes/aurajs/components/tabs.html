Tabs

Passing Data

There are two ways on how to pass data to the tabs component

  • Either by url
  • or by data

The JSON data provided for the tabs could look like this:

{% highlight javascript %} '{"id":"529f1f5b7783e","title":"Contacts","url":"details","items":[' + '{"title":"Details","action":"details","id":"529f1f5b777f1"},' + '{"title":"Permissions","action":"permissions","id":"529f1f5b7782f"}' + ']}' {% endhighlight %}

Options

  • url - url to get json data
  • data - alternatively passing data
  • instanceName - by defining an instanceName the name of triggered events can be affected
  • preselect (default: true) - if data.url is provided, the matching tabs element is preselected

Events

husky.tabs.<>.initialized

triggered when tabs are fully loaded

husky.navigation.<>.item.select[item object]

triggered when an item has been clicked
