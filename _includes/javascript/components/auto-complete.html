Auto-Complete

The Auto-Complete Field is a flexible field which helps the user to search for a relation.

{% endhighlight %}

  • abcdef

Usage

Via Javascript

{% highlight javascript %} var autoComplete = $('#auto-complete').huskyAutoComplete(options); {% endhighlight %}

Events

Event Name Description
auto-complete:loadData
{% highlight javascript %} autoComplete.data('Husky.Ui.AutoComplete').on('auto-complete:loadData', function() { ... }); {% endhighlight %}

Options

Option Name Type Default Description
url string null URL to load Data
valueName string "name" Property name for value
minLength integer 3 Control with up/down/enter key
value JSON object null Value to display at start
excludeItems JSON array [] This items would not be shown

Data

Request:

{% highlight javascript %} var url = '/contacts?search=abc'; {% endhighlight %}
{% highlight javascript %} { "total": 3, "items": [ { "id": "1", "name": "abcd" }, { "id": "2", "name": "abcde" }, { "id": "3", "name": "abcdef" } ] } {% endhighlight %}
