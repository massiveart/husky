Parameters:

  • [options] Object optional

    Configuration object

    • [url] String optional

      url to fetch data from

    • [instanceName] String optional

      enables custom events (in case of multiple tabs on one page)

    • [itemsRequestKey] String optional

      key with result-array for requested dropdown items

    • [appearance] String optional
    • [searchOptions] Object optional

      options to pass to search component

    • [groups] Object optional

      array of groups with id and align to specify groups to put items in

    • [small] Object optional

      if true the toolbar is displayed smaller

    • [hasSearch] Boolean optional

      if true a search item gets inserted in its own group at the end. A search item can also be added manually through the data

    • [searchAlign] String optional

      "right" or "left" to align the search if it's added automatically via the hasSearch option

    • [skin] String optional

      custom skin-class to add to the component

    • [showTitleAsTooltip] Boolean optional

      shows the title of the button only as tooltip

    • [data] Array optional

      if no url is provided

      • [title] String optional
      • [id] String optional
      • [disabled] Boolean optional
        is item disabled or enabled
      • [disableIcon] String optional
        icon in disable state
      • [iconSize] String optional
        large/medium/small
      • [class] String optional
        highlight/highlight-gray
      • [group] String optional
        id of the a group specified in the options
      • [position] Integer optional
        integer to sort the items - default 9000
      • [type] String optional
        if select, the selected item is displayed in main item (none/select)
      • [callback] Function optional
        callback function
      • [hidden] Boolean optional
        if true button gets hidden form the beginning on
      • [hideTitle] Boolean optional
        hide title from beginning
      • [items] Array optional
      • [items.title] String optional
      • [items.icon] String optional
        false will remove icon
      • [items.callback] Function optional
      • [items.divider] Boolean optional
        if true takes item as divider element
