DataGrid

(
  • [options]
)

Defined in husky_components/datagrid/main.js:1

Parameters:

  • [options] Object optional

    Configuration object

    • [data] Object optional

      if no url is provided (some functionality like search & sort will not work)

    • [resultKey] String optional

      the name of the data-array in the embedded in the response

    • [defaultMeasureUnit=px] String optional

      the unit that should be taken

    • [view='table'] String optional

      name of the view to use

    • [pagination=dropdown] Boolean | String optional

      name of the pagination to use. If false no pagination will be initialized

    • [paginationOptions] Object optional

      Configuration Object for the pagination

    • [viewOptions] Object optional

      Configuration Object for the view

    • [sortable] Boolean optional

      Defines if records are sortable

    • [searchInstanceName=null] String optional

      if set, a listener will be set for the corresponding search events

    • [columnOptionsInstanceName=null] String optional

      if set, a listener will be set for listening for column changes

    • [url] String optional

      url to fetch data from

    • [instanceName] String optional

      name of the datagrid instance

    • [preselected] Array optional

      preselected ids

    • [childrenPropertyName] Boolean | String optional

      name of the property which contains the number of children. False to indaticate that list is flat

    • [onlySelectLeaves] Boolean optional

      If true only the outermost children can be selected

    • [resizeListeners] Boolean optional

      If true a resize-listener will be instantiated, which is responsible for responsiveness

    • [contentFilters] String | Function optional

      Used for filtering data at a specifig attribute / column. If defined as callback the rows value will be passed. If defined as a string, an existing filter template will be applied (see matching types)

    • [matchings] Array optional

      configuration array of columns if fieldsData isn't set

      • [content] String optional
        column title
      • [width] String optional
        width of column (used by the table view)
      • [class] String optional
        css class of the column
      • [type] String optional
        type of the column. Used to manipulate its content (e.g. 'date')
      • [attribute] String optional
        mapping information to data (if not set it will just iterate through attributes)
